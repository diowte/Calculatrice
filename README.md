# Calculator (.NET MAUI)

Modern dark-themed calculator built with C# and .NET MAUI.

## Features
- Basic arithmetic operations (+, −, ×, ÷)
- Percentage and reciprocal (1/x)
- Clear (AC) and sign toggle (±)
- Custom rounded UI design

## Tech
C#, .NET MAUI, XAML

## Run
Open in Visual Studio 2022 and press F5.
